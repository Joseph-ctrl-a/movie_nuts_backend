Module

middlewareFactory

Middleware Factory Centralized access point for common middleware utilities used across the application. Returns a standardized set of middleware functions and helpers for Express apps.
Properties:
Name Type Description
logger function Request logging middleware.
auth function Authorization middleware that protects routes.
static Object Static file-serving utilities (from `global.createStaticOps()`).
json function JSON body parser middleware.
urlencoded function URL-encoded body parser middleware.
cors function CORS wrapper middleware for cross-origin requests.

View Source operations/middleware.js, line 7

Middleware utilities for Express setup.
Object
Example
const middleware = require('./factory/middlewareFactory')()
const app = express()

// Apply middlewares
app.use(middleware.cors)
app.use(middleware.json)
app.use(middleware.urlencoded)
app.use(middleware.logger({ showTime: true }))

// Use static serving
app.use('/public', middleware.static.serve('public'))
