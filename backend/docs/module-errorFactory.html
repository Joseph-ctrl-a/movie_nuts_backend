errorFactory

Error Handling Factory Provides a consistent and centralized way to generate, log, and throw HTTP and custom API errors in an Express app.

Error handling utilities
Object
Example
const error = require('./error')()

// Create an error object
const notFound = error.NotFound('User not found')

// Throw directly in a route
app.get('/users/:id', (req, res) => {
  const user = null
  if (!user) return error.throwNotFound(res)
})

// Custom error logging
error.log(new Error('Something went wrong'), 'UserController')
