# async connectDB() → {Promise.<void>}

MongoDB Connection Utility Establishes a connection to a MongoDB database using Mongoose. Wraps connection logic in a try/catch block and throws a custom `DatabaseConnectionError` on failure for consistent error handling.

  }

# async loadDatabase(connectDB, app) → {Promise.<void>}

Initializes the MongoDB connection and starts the Express server. Handles asynchronous database connection setup and safely starts the HTTP server. Logs success or failure messages and terminates the process gracefully on critical errors.
Parameters:
Name Type Description
connectDB function Function establishing a MongoDB connection.
app Express.Express Express application instance.

View Source loaders/loadDatabase.js, line 2

Promise.<void>
Example
const express = require('express');
const { connectDB } = require('./db/connectDB');
const loadDatabase = require('./loadDatabase');

const app = express();

(async () => {
  await loadDatabase(connectDB, app);
})();
